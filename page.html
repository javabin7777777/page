h1-h6 | p |strong

h1-h6: É o título da página e com tamanhos que vão de h1(o maior) ao h6(o menor).

Como usar: <h1> aqui o texto </h1> então fica assim:

aqui o texto

p : Esta tag é o parágrafo.O que tiver dentro terá formato de um parágrafo.

Como usar: <p> aqui o texto </p> então fica assim:

aqui o texto

strong: Esta tag coloca o texto em negrito.

Como usar:<strong> aqui o texto </strong> então fica assim: aqui o texto

ol | ul | li | a

ol: É uma lista ordenada,onde os ítens são dispostos de cima para baixa seguindo uma ordem, geralmente numérica.

Como usar: <ol> <li> aqui fica o ítem </li> </ol> então fica assim:

  1. Exemplo lista ordenada

ul: É uma lista não ordenada,onde os ítens são dispostos de cima para baixa.Diferentes marcadores marcam os ítens.

Como usar: <ul> <li> aqui fica o ítem </li> </ul> então fica assim:

  • Exemplo lista não ordenada

a: Tag âncora(achor) faz um "hyperlink",que através deste podemos navegar de uma página para outra. Seu atributo mais importante é o "href". Para todos navegadores as cores,por "default",para o "link" são:

  • azul: sublinhado e não visitado.
  • roxo ou lilás: sublinhado e visitado.
  • vermelho: sublinhado e ativo.

Como usar: <a href="aqui o endereço para onde se quer ir"> aqui nome para onde se quer ir</a> então fica assim: GooGle

font | abbr | del

font: Tag não mais suportada pelo "html 5",era utilizada no "html 4",com os atributos face(tipo de fonte);size(tamanhoda fonte);color(cor da fonte) no seu lugar ficou o uso de seletores,propriedades(ou atributos) e valores do "css".

Como era usado: <font face="" size="" color=""> aqui fica o texto &l/font>

abbr: Tag defini uma abreviação ou um acrônimo.O atributo "title" descreve a abreviação. de seletores,propriedades e valores do "css".

Como usar: <abbr title"aqui texto descritivo"> aqui abreviação ou acrônimo &l/abbr> ao passar o mouse sobre a abreviação,o texto descritivo aparece então fica assim: HTML

del: Tag defini um texto que será deletado do texto em questão.Geralmente os navegadores riscam o texto deletado com uma linha.

Como usar: <del> aqui o texto a ser deletado </del> então fica assim: Amanhã vou ao teatro
